# ScaleBridge Admin Portal

Admin portal for viewing contact form submissions from the ScaleBridge website.

## Features

- Secure admin login
- View all contact submissions
- Search and filter contacts
- Real-time data from MongoDB
- Responsive design

## Default Login Credentials

- Username: `scalebridge`
- Password: `31102003`

## Setup

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Deployment

Deploy to Vercel:

```bash
vercel
```

Make sure the backend API is deployed and accessible at the configured API_URL.
